# Airbnb Clone

A full-stack Airbnb clone web application built using Node.js, Express.js, MongoDB, EJS, Passport.js, and Cloudinary.

## Features

- User Authentication
- Create, Edit, Delete Listings
- Image Upload with Cloudinary
- Reviews and Ratings
- Flash Messages
- Responsive UI

## Tech Stack

- Node.js
- Express.js
- MongoDB
- EJS
- Bootstrap
- Cloudinary
- Passport.js

## Installation

```bash
npm install
npm start
```
